生成 github ssh 密钥

git使用4种协议来传输数据：本地传输，SSH 协议，Git 协议和 HTTP 协议。如果要对网络仓库进行写入操作，就必须使用ssh。而为电脑添加ssh key可以免去每次操作都输入密码的步骤。下面就向大家介绍如何为电脑添加ssh key。
